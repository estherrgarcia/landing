Transparent roadmap and Ecosystem collaboration

At source{d} we are creating a suite of Open Source tools enabling "Code as Data" and "Machine Learning on Code". From datasets and Machine Learning models to code retrieval and analysis tools, we have open-sourced more than 15 projects and are great believers in the value of Open Source transparency and collaboration.

Top OSS projects are:

go-git

is a highly extensible git implementation library written in pure Go. It can be used to manipulate git repositories at low level (plumbing) or high level (porcelain), through an idiomatic Go API.

Babelfish

Babelfish is a self-hosted server for source code parsing. It can parse any file, in any supported language, extracting an Abstract Syntax Tree (AST) from it and converting it into a Universal Abstract Syntax Tree (UAST).
